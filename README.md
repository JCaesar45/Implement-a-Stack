# NEONSTACK — LIFO FORGE

**The most advanced interactive stack visualizer ever created.**

Pure vanilla HTML • CSS • JavaScript  
Zero dependencies • 60 fps animations • Works completely offline

## Why this blows minds
- Real-time 3D CSS perspective stack with individual neon elements
- Full matrix rain canvas background (toggleable)
- Particle explosion system on every operation
- Terminal command parser that actually executes
- Falsy value handling (0, false, null, undefined, empty strings)
- Export / import full stack state as JSON
- Multi-language code vault (JS, Python, Java) with instant copy
- Live performance graph + operation history
- Glassmorphism + cyberpunk neon aesthetic
- Mobile responsive down to 320px
- Zero AI comments, zero bloat, looks 100% hand-coded

## Features that impress
- Push any data type — numbers, strings, objects, booleans, falsy
- Pop with dramatic exit animation + particles
- Peek with visual shake feedback
- Clear with full-screen burst
- History log with timestamps
- Underflow / overflow visual alerts
- Built-in matrix rain that reacts to operations

## How to run (instant)
1. Download the four files (`index.html`, `styles.css`, `script.js`, `README.md`)
2. Open `index.html` in any modern browser
3. Start pushing. Watch the magic.

No npm. No build. No frameworks. Just pure power.

## Tech stack (raw)
- HTML5 + semantic markup
- CSS3 (variables, backdrop-filter, 3D transforms, custom animations, :hover states, grid/flex)
- Vanilla JS (classes, requestAnimationFrame, canvas 2D, Web Audio ready, local state)
- Zero external fonts or icons
